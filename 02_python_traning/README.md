### Common Type Hints:

| Type | Example |
|------|---------|
| `int` | `age: int = 25` |
| `float` | `pi: float = 3.14` |
| `str` | `name: str = "Alice"` |
| `bool` | `is_active: bool = True` |

| `tuple` | `point: tuple[int, int] = (2, 3)` |
| `dict` | `data: dict[str, int] = {"a": 1, "b": 2}` |
